# 项目规则

- 在任何带选中态或切换态的 Anki 题卡模板中，前面必须默认未选中，且背面在读取当前作答后必须立即清理该题的缓存状态，禁止把上一次作答的选中态带到下一次复习；判断题、单选题、多选题及后续同类模板都按此规则实现和回归验证。
- 在任何带输入态的 Anki 题卡模板中，背面必须基于用户刚才的输入做逐字符判题反馈，并在读取结果后立即清理该题缓存，禁止把上一次输入带到下一次复习；输入挖空卡、严格输入题及后续同类模板都按此规则实现和回归验证。
- 后续修改 Anki 笔记模板时不要主动产出或重建 apkg，默认直接使用 AnkiConnect API 更新现有笔记类型模板，仅在用户明确要求导出 apkg 时才生成产物。
