# anki-templates

这个目录存放 Anki 模板资源。

当前包含：

- `basic-enhanced`
  - Basic 风格的简洁增强模板
- `cloze-enhanced`
  - Cloze 风格的简洁增强模板
- `input-cloze`
  - 可在原句挖空位置直接输入的 Cloze 题类
- `code-completion`
  - 代码补全题模板，支持多处挖空和语言高亮
- `input-exact`
  - 单行 / 多行严格输入题模板
- `single-choice`
  - 单选题模板
- `multiple-choice`
  - 多选题模板
- `true-false`
  - 判断题模板

每个子目录通常包含：

- `front.html`
- `back.html`
- `style.css`
- `README.md`

使用方式：

1. 打开对应子目录
2. 按目录内 `README.md` 的说明，把模板内容粘到 Anki
